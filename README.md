# SquareUp
A modern, responsive landing page built from a professional Figma design. This project serves as a demonstration of clean, semantic HTML5, modular CSS3 architecture, and "pixel-perfect" implementation.

Tech Stack

HTML5: Semantic markup for SEO and accessibility (header, main, footer, section).
CSS3: Use of Flexbox, CSS Variables (:root), and native smooth scrolling.
Modular Architecture: The project is broken down into specific CSS files for better maintainability and scalability:  reset.css — Standardizes styles across different browsers.
fonts.css — Local font integration.style.css — Global styles and design tokens (variables).
footer_homepage.css — Component-specific styling.
mq_homepage.css — Dedicated file for responsive design and media queries.  

Key Features

Pixel Perfect: Precise implementation following the original Figma layout.

Modular CSS: Structured file system that prevents "spaghetti code" and allows for easy updates.  

Responsive Design: Fully optimized for desktop, tablet, and mobile devices using a structured media query system.  

Smooth Navigation: Integrated native CSS smooth-scroll behavior for a premium user experience.

Optimized Performance: Clean code without unnecessary heavy libraries or frameworks.

Project Structure

├── css/

│   ├── fonts.css           # Typography

│   └── mq_homepage.css     # Media queries (Mobile/Tablet)

### My Goals with this Project
I built this project to practice **modern CSS organization**. My focus was on solving common layout challenges using Flexbox and ensuring that the cascade remains predictable by separating global styles from component-specific ones[cite: 13, 15].
